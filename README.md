# CurrencyClean

Минималистичный конвертер валют для iOS на SwiftUI. Написан с использованием современного стека и чистой архитектуры. 

## Особенности
- **Архитектура:** MVVM.
- **Сетевой слой:** Асинхронная загрузка данных (Async/Await) без использования сторонних библиотек.
- **Интерфейс:** Декларативная вёрстка на SwiftUI с поддержкой динамического обновления данных.
- **UX:** Обработка состояний загрузки (ProgressView) и локальная заглушка на случай сбоев сети.

## Технологический стек
- Swift 5.10+ / iOS 17+
- SwiftUI
- Combine (ObservableObject / Published)
- URLSession (Swift Concurrency)
- JSONDecoder

## Как запустить
1. Склонируйте репозиторий.
2. Откройте `CurrencyClean.xcodeproj` в Xcode.
3. Выберите любой симулятор (например, iPhone 15) и нажмите `Cmd + R`.
